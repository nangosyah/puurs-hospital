## Puurs Hospital Emergency Department Dashboard

An interactive analytics dashboard built with Python Dash for monitoring Emergency Department performance at Puurs Hospital. 

The dashboard tracks key operational metrics including staffing levels, patient wait times, door-to-doctor intervals, length of stay, LWBS rates, and clinical service utilization.
